## Deployment Instructions

### Netlify
1. Go to https://app.netlify.com/drop
2. Upload this ZIP file
3. Your site will go live instantly

### Vercel
1. Go to https://vercel.com
2. Create new project → upload folder
3. Deploy

### GitHub Pages
1. Create new repo
2. Upload all files
3. Go to Settings → Pages → Deploy from root
